<h1> ToDoApp </h1>
A clean, minimal to-do app to manage tasks and track progress while staying organized.
<hr> 
<h4 style="margin-bottom:0;">Tech Stack:</h4>
<ol>
<li> Backend : NodeJs </li>
<li> Frontend : Vite + React </li>
<li> Database : FireBase + NoSQL </li>
</ol>
<hr> 
<h4 style="margin-bottom:0;">Intended Features:</h4>
<ul>
<li> Add remove edit and filter tasks intuitively </li>
<li> Save data per user in backend based on login/signup </li>
<li> Securing data so it cannot be leaked </li>
</ul>
